【仕様書】商品在庫管理システム

■ 概要
倉庫担当者が使う商品の在庫管理画面を作成する。
在庫数が一定以下になった商品を一覧で強調表示する。

■ テーブル設計
テーブル名: products
- id
- code: string(50) 必須 ユニーク
- name: string(255) 必須
- category: string(100) 必須
- price: integer 必須 デフォルト0
- stock: integer 必須 デフォルト0
- alert_threshold: integer 必須 デフォルト10
- is_active: boolean デフォルトtrue
- description: text nullable
- created_at / updated_at

■ 画面一覧
1. 一覧画面
   - テーブル形式で表示
   - stock が alert_threshold 以下の行は背景を赤くする
   - is_activeがfalseの行はグレーアウト
   - 絞り込み: categoryのselectボックス（一覧上部に設置）

2. 詳細画面
   - 全カラムを表示
   - 在庫状況（正常 / 要補充 / 停止中）をラベルで表示

3. 新規登録・編集画面
   - is_activeはチェックボックス
   - categoryは以下固定値のselect
     食品 / 日用品 / 電化製品 / 衣類 / その他

■ バリデーション
- code: required|string|max:50|unique:products（編集時自分除外）
- name: required|string|max:255
- category: required|in:食品,日用品,電化製品,衣類,その他
- price: required|integer|min:0
- stock: required|integer|min:0
- alert_threshold: required|integer|min:0
- is_active: boolean
- description: nullable|string

■ Controller追加要件
- 一覧のcategory絞り込みはControllerでwhere処理
- is_activeの未チェック時は0として保存

■ 命名規則
- Model: Product
- Controller: ProductController
- Request: StoreProductRequest / UpdateProductRequest
- ルート: resource（products）
- View: resources/views/products/
